# Check
###### Check if a string (first argument, str) ends with the given target string (second argument, target)

### Requirenments
* NodeJS >= 4.x

### Install
```sh
$ git clone git@github.com:rowen92/check.git
$ npm install
```

### Run in development mode
```sh
$ npm start
```

### Testing
```sh
$ npm test
```
